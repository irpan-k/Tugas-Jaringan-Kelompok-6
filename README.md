## ⚙️ Spesifikasi Perangkat & Simulator
Untuk mereplikasi atau meninjau topologi jaringan dalam proyek ini, kami menggunakan spesifikasi lingkungan berikut:
* **Simulator Jaringan:** GNS3 v2.2.59
* **Router Cisco IOS Image:** `c3725-adventerprisek9-mz.124-15.T14.image`
* **End-Host Perangkat:** VPCS (Virtual PC Simulator)

> 📌 **Catatan:** Optimalisasi penggunaan VPCS dilakukan untuk menghemat penggunaan RAM dan CPU pada perangkat laptop anggota kelompok tanpa mengurangi fungsi pengujian Layer 3 (Ping & Traceroute).
